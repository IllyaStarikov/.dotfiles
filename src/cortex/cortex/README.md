# Cortex Core Package

Python package for the Cortex local AI assistant.

## Modules

- `cli.py` - Command-line interface
- `core.py` - Core AI interaction logic
- `config.py` - Configuration management
- `health.py` - Health check utilities
- `statistics.py` - Usage statistics tracking
- `system_utils.py` - System utility functions
- `providers/` - AI provider implementations (MLX, Ollama, etc.)

## Usage

```bash
cortex              # Start interactive session
cortex status       # Check AI server status
cortex model <name> # Switch AI model
```

See parent `README.md` for full documentation.
