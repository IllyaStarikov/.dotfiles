# WezTerm Theme Files

Lua theme configuration files for WezTerm terminal.

## Files

- `tokyonight_day.lua` - Light theme
- `tokyonight_night.lua` - Dark blue theme
- `tokyonight_moon.lua` - Purple-tinted dark theme
- `tokyonight_storm.lua` - Balanced dark theme

## Usage

Active theme is loaded via `~/.config/wezterm/theme.lua` which is generated by the theme-switcher.

See `src/theme-switcher/README.md` for switching themes.
