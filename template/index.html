Installation

One-Command Setup

git clone https://github.com/IllyaStarikov/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles && ./src/setup/setup.sh

This will install all dependencies and create necessary symlinks automatically.

Step-by-Step Installation

# 1. Clone the repository
git clone https://github.com/IllyaStarikov/.dotfiles.git ~/.dotfiles

# 2. Install core dependencies (macOS)
brew install neovim tmux ripgrep fd eza bat delta

# 3. Create symlinks
~/.dotfiles/src/setup/symlinks.sh

# 4. Install Neovim plugins
nvim --headless "+Lazy! sync" +qa

# 5. Install tmux plugins
~/.tmux/plugins/tpm/bin/install_plugins

Try with Docker

# Build the Docker image
docker build -t dotfiles https://github.com/IllyaStarikov/.dotfiles.git

# Run interactive shell
docker run -it dotfiles

Perfect for testing without affecting your system.
